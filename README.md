# hf-gguf-finder — migrated

This project has been split into two dedicated repositories under the
[gguf-run](https://github.com/gguf-run) organization:

| Tool | Repo | Description |
|------|------|-------------|
| **finder** | [github.com/gguf-run/finder](https://github.com/gguf-run/finder) | Search Hugging Face Hub for GGUF files |
| **gguf-run** | [github.com/gguf-run/gguf-run](https://github.com/gguf-run/gguf-run) | Search, download, and run with llama.cpp |

### Quick install

```bash
go install github.com/gguf-run/finder@latest
go install github.com/gguf-run/gguf-run@latest
```
